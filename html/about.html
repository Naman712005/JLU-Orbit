JLU Orbit

Project & Developers

About the project

JLU Orbit is a final-year engineering project built for the Faculty of Science & Technology at Jagran Lakecity University. It brings together a LinkedIn-style professional network with a developer-first blogging and research hub, designed specifically for students, faculty, and project collaborators.

Goals

  • • Central hub for projects, research and groups
  • • Encourage open, engineering-focused knowledge sharing
  • • Prepare students for modern developer workflows

How it was built

The system follows a classic web application architecture with a Node.js backend, REST APIs and a modern frontend built using HTML, Tailwind CSS, and vanilla JavaScript. Socket.IO is used to enable realtime notifications for user interactions.

Backend

  • • Node.js / Express
  • • RESTful API design
  • • JWT-based authentication
  • • MongoDB

Frontend

  • • HTML5 with modular views (feed, profile, groups, research)
  • • Tailwind CSS for responsive layouts
  • • Vanilla JavaScript for all interactions
  • • Font Awesome for iconography

Key features

  • • Community feed with posts, likes, comments and shares
  • • Profile pages with academic and project context
  • • Groups for project teams, hackathons and study circles
  • • Research hub for abstracts, content and keywords
  • • Search across posts and contributors
  • • Realtime notifications for activity

Technologies used

Node.js / Express MongoDB (NoSQL) Socket.IO Tailwind CSS Vanilla JavaScript HTML5
